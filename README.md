# 人肉 Transformer · 京东金融 JDS AI 培训教具

> 把 Transformer 的核心原理做成**可上手演练的教具**——让没有技术背景的同学，也能在一节课里真正 *get* 到注意力机制（Attention）到底在干什么。

这是我为**京东金融 JDS（校招生）AI 培训体系**设计的教学材料。其中"人肉 Transformer"教学法是我的一次中国首创尝试：不用一行公式，让学员用身体和卡片"扮演" Transformer 的每一步，亲手跑一遍注意力的计算流程。多场培训实测下来，非技术同学也能抓住模型运转的内核。

## 🎯 设计理念

整套 JDS AI 课程，我想解决的是同一个问题：**怎么让"原理"变成"手感"。**

- **从原理讲起**：AI 到底是怎么工作的，而不是只教工具怎么点。
- **重点讲提示词设计**：理解了技术原理之后，如何"操纵" AI 来完成各种任务。
- **人肉 Transformer**：把最难的 Attention 机制，拆成人人能演的一套动作。
